== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'bdtaskcrm'
3. Activate bdtaskcrm from your Plugins page.




= From WordPress.org =

1. Download Bdtaskcrm Plugin.
2. Upload the 'Bdtaskcrm' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Bdtask 'Bdtaskcrm' from your Plugins page.
